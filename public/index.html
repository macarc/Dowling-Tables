Dowling Tables

Memorise the Latin declensions online

This site will help you to memorise the Latin tables from Wheelock necessary for the Dowling method.

It was inspired by Dowling's Wheel - however, I wanted to have something supporting macrons, which can be useful to learn.

How to use

Choose one of the options below to study.

It's recommended that you write them out by hand a few times first, before using the computer.

You can type macrons by pressing Shift + [vowel key] (e.g. Shift + a makes ƒÅ).
